```bash
⏺ macOS에서 OpenGL 코드 실행하기                                                                            
                                                                                                            
  고친 것                                                                                                   
  ┌──────┬────────────────────────────────────────┬─────────────────────────────────────────────────┐
  │ 항목 │             원본 (Windows)             │                      macOS                      │       
  ├──────┼────────────────────────────────────────┼─────────────────────────────────────────────────┤       
  │ 헤더 │ #include <GL/glew.h>                   │ 제거                                            │
  ├──────┼────────────────────────────────────────┼─────────────────────────────────────────────────┤
  │ 헤더 │ #include <GL/freeglut.h>               │ #include <GLUT/glut.h> + #include <OpenGL/gl.h> │
  ├──────┼────────────────────────────────────────┼─────────────────────────────────────────────────┤
  │ GLEW │ glewInit() 등 사용                     │ 전부 제거 (기본 OpenGL 함수만 쓰므로 불필요)    │
  ├──────┼────────────────────────────────────────┼─────────────────────────────────────────────────┤
  │ main │ void main                              │ int main                                        │
  ├──────┼────────────────────────────────────────┼─────────────────────────────────────────────────┤
  │ 종료 │ glutLeaveMainLoop()                    │ exit(0)                                         │
  ├──────┼────────────────────────────────────────┼─────────────────────────────────────────────────┤
  │ 옵션 │ glutSetOption(...), glutCloseFunc(...) │ 제거 (freeGLUT 전용 함수)                       │
  ├──────┼────────────────────────────────────────┼─────────────────────────────────────────────────┤
  │ 경고 │ -                                      │ #define GL_SILENCE_DEPRECATION 추가             │
  └──────┴────────────────────────────────────────┴─────────────────────────────────────────────────┘
  사용 파일

  - SimplefreeGLUTcode.cpp — 소스 코드 (이것만 있으면 됨)
  - .sln, .vcxproj — Windows Visual Studio 전용이라 macOS에선 사용 안 함

  컴파일 & 실행

  g++ SimplefreeGLUTcode.cpp -o SimplefreeGLUTcode -framework OpenGL -framework GLUT
  ./SimplefreeGLUTcode

  -framework OpenGL -framework GLUT은 macOS에 기본 내장된 라이브러리라 별도 설치 없이 바로 사용 가능합니다.

```
